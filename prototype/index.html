Visual tone

Warm, practical, and food-native.

The system avoids clinical fitness UI. It uses natural neutrals, compact radius, thin borders, and restrained shadows.

Foundation logic

Small primitives, clear semantics.

Primitive colors feed semantic groups, semantic groups feed components, and every value remains traceable to a CSS variable.

Color Tokens

4 main colors, ordered by family.

The showcase treats the palette as four product families: Neutral, Olive, Tomato, and Amber. Each family lists its variables in order, from softer values to stronger values. Support aliases stay explicit below, but no longer read as primary colors.
