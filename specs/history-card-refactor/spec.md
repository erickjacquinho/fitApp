# Feature Specification: HistorySessionCard Refactor
Refatorar o componente `HistorySessionCard.tsx` para simplificar a interface de usuário.
- Remover a exibição do `programName` e do progresso de `completedSets/totalSets`.
- Adicionar ícones da biblioteca `lucide-react-native` (Calendar, Clock) para identificar os metadados de data e duração.
- Otimizar o layout usando flex-wrap para melhor responsividade e escaneabilidade.
